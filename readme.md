###### Comandos usados para configurar Docker:

`touch Dockerfile` para crear el archivo.

`touch .dockerignore` para crear el archivo ignore.

`nano  Dockerfile` y `nano  .dockerignore` para editar los  archivos.

`docker build -t countries-game .` para crear la imagen.

`docker run -d -p 8080:80 --name countries-game-container countries-game` para probar la app.
